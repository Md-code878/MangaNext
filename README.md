# MangaNext Server
A lightweight Manga server built in Kotlin for Termux.

## Features
- **Mihon Extension Support**: Load .apk extensions directly.
- **Kotatsu UI**: Optimized mobile-web interface.
- **Backup/Restore**: Supports `.tachidesk` and `.zip`.
- **Lightweight**: Optimized to run on Android via Termux.

## Setup
1. `./gradlew build`
2. `./gradlew run`
